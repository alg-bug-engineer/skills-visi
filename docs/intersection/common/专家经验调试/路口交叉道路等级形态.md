不同等级道路相交，形成了不同重要程度的路口：

主干路-主干路相交：城市路网中最关键的节点，交通流量巨大，是交通管理的重中之重。

主干路-次干路相交：重要的交通转换节点，需保证主流向的畅通。

次干路-次干路相交：区域性的交通节点，流量适中。

主干路/次干路-支路相交：需要明确主次，保证主路优先。

支路-支路相交：最低等级的路口，服务局部微循环。

路口的几何形状直接决定了交通流的交织与冲突模式。常见的形态包括：

十字路口：两条道路垂直相交，形式简单，组织方便，是最基本、最常见的路口形式。

T型路口（丁字路口）：一条路的尽端与另一条路的中部相接，冲突点相对较少。

Y型路口：三条道路以锐角或钝角相交，视距和转弯半径是设计难点。

X型路口：两条道路斜向交叉，锐角处视距不良，对左转车辆不利。

环岛：路口中心设岛，车辆绕行，能消除冲突点，适合多路交汇处，但占地大、通行能力有限。

畸形路口：包括错位、多路（五叉及以上）等不规则路口，交通组织最为复杂，冲突点数量会急剧增加，通常需要特殊的交通工程手段来改善。
