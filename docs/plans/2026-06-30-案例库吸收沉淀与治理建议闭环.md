# 案例库吸收沉淀与治理建议闭环 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 新增可浏览「案例库」（行业案例+路口案例），用右下角浮层 Toast 替代经验吸收卡片并显式反馈认知/诊断经验的去重吸收结果，治理建议借助大模型深度融合经验/案例/溯源并标注可溯源、可跳转的依据，形成完整闭环。

**Architecture:** 全部复用现有基础设施。后端：增强 `CaseLibraryService` 解析、新增 `/cases/*` 路由、`IntersectionProfileStore` 写方法返回去重结果、`SuggestionResult.references` 派生、强化建议 prompt 融合。前端：`UnderstandingProcessPanel` 增「案例库」Tab + `CaseLibraryPanel.vue`、删除 `ExperienceAbsorptionPanel`、新增 `ExperienceAbsorptionToast.vue` + `useAbsorptionToasts.ts`、建议卡参考依据 chip。

**Tech Stack:** Backend FastAPI + Pydantic + pytest（`uv run pytest`）。Frontend Vue 3 + TS + Vitest（`npm run test:unit`）。
